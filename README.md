# uniswap-defi-data-platform
I enjoy learning how the Blockchain technology works, and how DeFi born from this new technology.
I've always wanted to understand how one of the biggest protocols in DeFi works behind the scenes, because not everything is what you see at the UX. 
This proyect is going to be an MVP that'll take me around 3 months of work.
The scope is:
+ Retrieve the data directly from Uniswap Subgraphs APIs, not external tools.
+ Build a Medallion Architecture of the data
+ Use Unity Catalog across the proyect
+ Leverage Delta Lake to store efficienly, implementing some kind of features like vacuums or z-orders.
